This release focuses on quality improvements across content editing, validation, and previews.

## Highlights

- Smart rename support now helps keep filenames in sync with front matter data.
- Front matter schema validation for markdown files is now supported and can be disabled with the `frontMatter.validation.enabled` setting.
- Preview support now works with the integrated VS Code browser.
- The chatbot command and related functionality have been removed.
- Sponsor AI setting support via `frontMatter.sponsors.ai.enabled` has been removed.
